>
> 在线体验：https://personal-workspace.onrender.com

---

## 一、项目概述

个人工作台是一站式效率工具，集成待办管理、番茄专注、日历日程、备忘录、工具聚合五大核心模块，并在此基础上设计了游戏化自律系统"成长花园"。项目从产品需求定义到工程实现到用户反馈迭代，全程实践 AI 驱动开发的工作流。

**核心理念**：用产品思维解决个人生产力问题，以正反馈机制帮助用户建立自律习惯。

---

本项目是"AI PM"工作方式的完整实践案例。从 PRD 到部署上线，AI 深度参与全链路：

| 阶段 | 工具 | 方法 | 产出 |

## 三、产品功能架构

### 3.1 核心模块（原型阶段完成，v2.28）

| 模块 | 功能描述 |
|------|----------|
| 待办管理 | 多优先级（高/中/低/日常）、多日面板、持续待办跨日复制、子任务系统 |
| 番茄专注 | 工作/休息循环、环形进度条、多提示音、浏览器后台计时精准 |
| 日历日程 | 日期栏快捷切换、日程自动创建每日待办 |
| 备忘录 | 图片上传与双击放大、拖拽排序 |
| 工具聚合 | 自定义快捷方式、文件夹分组管理 |

### 3.2 深度迭代（v2.29+）

**全局待办视图**：打平所有日期待办，统一列表排序，支持日期范围筛选与隐藏已完成，解决"只看今天"的视野局限。

**周报系统**：自动汇总每周番茄钟数、待办完成（按优先级分布）、日程、Git 提交记录，支持手动编辑总结和下周计划，新周自动弹窗提醒。

**交互打磨**：
- 子任务拖拽排序（解决父级拖拽冲突、Firefox 兼容）
- Tab 键循环切换优先级（中→高→低→中）
- Ctrl+Enter 多行文本输入
- 番茄钟跳过休息

### 3.3 游戏化自律系统「成长花园」

参考 Habitica（RPG 任务机制）、Forest（专注种树）、Duolingo（连胜+成就）三款成熟产品，设计五大核心机制：

**XP 经验值与等级成长**
- 完成待办 +10~15 XP，完成番茄钟 +25 XP，完美一天 +50 XP
- 等级公式：N 级需要累计 `100 × N × (N+1) / 2` XP
- 渐进式难度曲线，保持长期激励

**每日任务**
- 4 个动态任务（完成 3 个待办 / 2 个番茄钟 / 1 个高优 / 所有日常）
- 每天早上自动刷新，回溯已有数据计算进度
- 全部完成触发"完美一天"

**连胜系统**
- 连续完美天数累计连胜，断连重置
- 连胜天数在游戏面板中醒目展示

**成就徽章**
- 17 个成就分四类：首次类（3）、累计类（6）、连胜型（3）、等级型（4）
- 传奇级成就（如"任务传说"完成 500 待办、"传奇连胜"连续 100 天）用全屏弹窗庆祝
- 普通成就不打断操作，Toast 提示 3 秒

**设计原则**：纯正反馈，无惩罚——只奖励不扣分，低产出日也能积累 XP，确保用户心理安全感。

### 3.4 产品架构亮点

- **事件驱动松耦合**：游戏系统通过 `window.notifyGame()` 与待办、番茄钟模块集成，无需修改核心业务逻辑
- **数据持久化**：LocalStorage 存储，跨天自动状态重置（每日任务刷新、跨天检测）
- **模块化架构**：~10 个独立 JS 文件，职责清晰，便于并行迭代

## 四、迭代方法与数据闭环

### 迭代流程

```
用户反馈收集 → 问题识别与优先级排序 → 方案设计 → 
Git 分支开发 → 测试验收 → 合并主线 → 更新文档
```

每次迭代在独立 Git 分支进行，测试满意后合并至 master。累计完成 20+ 次功能迭代，全部使用中文语义化 Commit Message。

### 关键迭代决策

- **子任务从 input 改为 textarea**：解决多行输入需求，同时保持 Ctrl+Enter 换行、普通 Enter 完成的快捷操作习惯
- **Tab 键选择优先级**：观察到自己频繁手动切换下拉框，通过 Tab 循环将切换成本降为零
- **番茄钟跳过休息**：在某些场景下用户不想等待休息，提供一键跳过而非强制等待
- **全局待办视图**：多日面板查看效率低，打平所有日期统一管理

这些决策体现了"发现使用摩擦 → 最小改动解决"的产品迭代思维。

## 五、技术实现

| 层 | 技术选型 | 选型理由 |
|------|----------|----------|
| 前端 | 原生 JavaScript（无框架） | 轻量级项目，无需框架开销；AI 工具对原生 JS 生成质量高 |
| 后端 | Node.js + Express | 成熟稳定，部署成本低 |
| 数据存储 | LocalStorage + JSON 文件 | 离线可用，零配置 |
| 部署 | Render 免费托管 | 快速上线 Demo，面向用户收集反馈 |
