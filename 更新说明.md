# 个人工作台 v2.29 更新说明

## 修复内容

### 1. 拖拽虚框边界检查 ✅
- 拖拽时严格检查鼠标是否在容器内
- 拖拽元素跑到容器外时自动移回

### 2. 隐藏按钮问题修复 ✅
- 给 `shortcut-item` 添加 `position: relative` 和 `overflow: hidden`
- 编辑/删除按钮不再溢出到错误位置

### 3. 文件夹工具预览模式UI优化 ✅
- **1-2个工具**：横向1行2列，居中显示，文件夹高度较矮
- **3-4个工具**：2×2网格填满，文件夹高度较高
- 图标大小：2.2rem，更醒目
- 文件夹名称：0.9rem 加粗，有分隔线

### 4. 按钮位置调整 ✅
- 添加和新建文件夹按钮放在一起
- 顺序：添加 → 新建文件夹

### 5. 其他修复 ✅
- 移除多余的手机图标
- 删除按钮增大
- 全局禁止文本选择，解决I型光标问题
- 背景路径修复
