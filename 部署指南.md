# 🚀 个人工作台 - 部署指南

## 推荐方案：Render 全栈部署 (免费)

### 第一步：准备 GitHub 仓库
1. 注册 GitHub 账号（如果没有的话）
2. 创建一个新仓库（命名为 `personal-workspace` 或其他你喜欢的名字）
3. 把这个项目推送到 GitHub

### 第二步：部署到 Render

1. **访问 Render**：https://render.com
2. **注册账号** - 使用 GitHub 账号注册最简单
3. **创建新 Web Service**
   - 点击 "New +" → "Web Service"
   - 选择你的 GitHub 仓库
   - 配置：
     - Name: `personal-workspace` (你的项目名称)
     - Region: 选择离你近的 (如 Singapore)
     - Branch: `main` 或 `master`
     - Root Directory: 留空 (项目在根目录)
     - Runtime: `Node`
     - Build Command: `npm install`
     - Start Command: `npm start`
4. **选择免费套餐** - 点击 "Create Web Service"

### 第三步：等待部署完成！
- Render 会自动部署你的项目
- 部署成功后，你会获得一个 URL，类似 `https://your-project-name.onrender.com`
- 这个 URL 就是你可以发给 HR 的作品集地址！

## 💡 注意事项
- 免费版会在 15 分钟无访问后休眠，但重新访问会自动启动
- 文件上传在免费版会重启后丢失（可展示功能即可）
- 数据会在重启后重置为默认值（这个是正常的，展示功能）
