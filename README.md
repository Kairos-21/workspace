# 个人工作台 v2.29

**一站式效率工具**，集成待办管理、番茄专注、日历日程、游戏化自律系统，旨在用产品思维解决个人生产力问题。

🔗 在线地址：`https://personal-workspace.onrender.com`

---

## 核心功能

### 📝 待办管理
- 多优先级（高/中/低/日常）+ 多日任务面板
- **持续待办**：未完成自动跨日复制，保持子任务状态
- **全局待办视图**：打平所有日期，支持日期筛选 + 隐藏已完成
- **子任务系统**：多行文本编辑、拖拽排序、完成状态跟踪
- Tab 键快速切换优先级，Ctrl+Enter 多行输入

### 🍅 番茄专注
- 工作/休息自动循环，浏览器后台标签页计时精准
- 可跳过休息直接开始下一轮
- 多提示音可选，可视化环形进度条

### 🎮 成长花园（游戏化自律）
- **XP 经验值 & 等级系统**：做事赚 XP，持续升级
- **每日任务**：4 个动态任务（完成待办/番茄钟/高优/日常），自动刷新
- **连胜系统**：完美一天达成后累计连胜，断连重置
- **成就徽章**：17 个成就分四类（首次/累计/连胜/等级），传奇级弹窗展示
- 设计参考 Habitica / Forest / Duolingo，纯正反馈无惩罚

### 📊 周报
- 自动汇总每周番茄钟数、待办完成（按优先级分布）、日程、Git 提交
- 支持手动编辑总结和下周计划，历史周报可切换查看

### 📅 日历日程 & 备忘录
- 日期栏快速切换，日程自动创建每日待办
- 备忘录支持图片上传/双击放大/拖拽排序

### 🔗 工具聚合
- 自定义快捷方式 + 文件夹分组管理

## 技术栈

| 层 | 技术 |
|---|------|
| 前端 | 原生 JavaScript（模块化架构，~10 个功能模块） |
| 后端 | Node.js + Express |
| 数据 | LocalStorage 为主 + JSON 文件存储 |
| 部署 | Render 全栈托管 |

## AI 驱动的开发实践

本项目全程使用 AI 工具链辅助，践行"AI PM"工作流：

| 阶段 | 工具 | 产出 |
|------|------|------|
| 需求定义 | Gemini（多轮对话 + 纠错） | PRD（产品需求文档） |
| 方案设计 | Gemini → Cursor | UI 设计文档 + 技术实施文档 |
| 工程实现 | Cursor + Trae | 前后端代码、CSS 样式 |
| 方向验证 | Coze | AI 工作助手原型 |

从 PRD 到部署上线，AI 深度参与全链路，极大压缩开发周期。
