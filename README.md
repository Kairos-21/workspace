# 个人工作台

一站式效率工具，集成待办管理、番茄专注、日历日程、游戏化自律系统。独立开发，AI 驱动全流程。

🔗 **在线体验**：[personal-workspace.onrender.com](https://personal-workspace.onrender.com)

---

## 功能模块

### 待办管理
- 多优先级（高/中/低/日常）+ 多日任务面板
- 持续待办：未完成自动跨日复制，保持子任务状态
- 全局待办视图：打平所有日期，按优先级筛选，隐藏已完成
- 子任务系统：多行文本编辑、拖拽排序、完成状态跟踪
- 置顶功能：重要任务固定顶部，橙色边框高亮
- 快捷键：Tab 切换优先级、Ctrl+Enter 多行输入、Ctrl+↑↓ 排序

### 番茄专注
- 工作/休息自动循环，浏览器后台标签页计时精准
- 可跳过休息直接开始下一轮，时间补录功能
- 多提示音可选，可视化环形进度条
- 关闭页面自动暂停，冷启动恢复状态

### 成长花园（游戏化自律）
- XP 经验值 + 等级系统：做事赚 XP，渐进式难度曲线
- 每日任务：4 个动态任务，自动刷新回溯
- 连胜系统：完美一天累计连胜，断连重置
- 成就徽章：17 个成就分四类，传奇级全屏弹窗
- 设计原则：纯正反馈，无惩罚

### 周报
- 自动汇总：番茄钟数、待办完成（按优先级分布）、日程、Git 提交
- 跳过休息次数 & 最长连续工作统计
- 手动编辑总结和下周计划，历史周报可切换

### 日历 & 备忘录
- 日期栏快速切换，日程自动创建每日待办
- 备忘录支持图片上传、双击放大、拖拽排序

### 工具聚合
- 自定义快捷方式 + 文件夹分组管理，支持拖拽排序

## 技术栈

| 层 | 技术 |
|---|------|
| 前端 | 原生 JavaScript（无框架，模块化架构） |
| 后端 | Node.js + Express |
| 存储 | LocalStorage + JSON 文件 |
| 部署 | Render 全栈托管 |

## AI 驱动的开发实践

本项目全程使用 AI 工具链辅助，践行"AI PM"工作流：

| 阶段 | 工具 | 产出 |
|------|------|------|
| 需求定义 | Gemini | PRD |
| UI 设计 | Gemini → Cursor | UI 设计 + 技术实施文档 |
| 工程实现 | Cursor + Trae | 前后端代码 |
| 方向验证 | Coze | AI 工作助手原型 |

从 PRD 到部署上线，AI 深度参与全链路。2 天完成原型开发与上线，后续 20+ 次迭代持续打磨。
